# Enterprise AI Business Assistant 💼

An advanced RAG (Retrieval-Augmented Generation) and Multi-Step Sales Analytics Agent built with Streamlit, FAISS, and Groq (Llama 3.1).

## 🚀 Features
- **Document Knowledge Base:** Upload PDFs and query them using FAISS L2 vector distance matching.
- **Sales Data Analysis:** Interactive Plotly dashboards combined with a multi-step analytical LLM agent.
- **Executive Automation:** Synthesizes multiple data sources into cohesive weekly business reports.

## 🛠️ Tech Stack
- **Frontend:** Streamlit
- **Embeddings & Vector Store:** SentenceTransformers (`all-MiniLM-L6-v2`), FAISS
- **LLM Engine:** Groq API (`llama-3.1-8b-instant`)
- **Data Visualization:** Pandas, Plotly Express
